# 🎓 Placement Predictor App

The **Placement Predictor App** is a Machine Learning web application that predicts whether a student will get placed based on their **CGPA and IQ** using a **Logistic Regression** model. It follows a complete ML pipeline including data preprocessing, feature selection, train-test split, scaling using `StandardScaler`, model training, and evaluation. The app provides an interactive interface built with **Streamlit**, where users can input values via sliders and get real-time predictions. It also includes **data visualization (scatter plot)** and a **decision boundary plot** to understand how the model separates placed vs non-placed students.

👉 **Live App:** [https://placement-project-logreg.onrender.com/](https://placement-project-logreg.onrender.com/)

---

## 🚀 Tech Stack

* Python, Pandas, NumPy
* Scikit-learn (Logistic Regression)
* Matplotlib, Mlxtend
* Streamlit (UI)
* Render (Deployment)
